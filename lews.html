Landslide Early Warning System (LEWS)

LEWS is a low cost MEMS-based monitoring and early warning system designed to challenge the expensive conventional monitoring systems. LEWS consists of four processes:

  1. Sensing
  2. Data Warehousing and Scrutinizing
  3. Prediction
  4. Alerts

Below is an animated version of LEWS for a better understanding.

We programmed a Microcontroller Board (Arduino Mega 2560) to receive data from different sensors and then get it transferred via a SIM card enabled GSM Module. The data is collected in our cloud-based server every 11 minutes. You can track real-time landslide updates on this website.
